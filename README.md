## 📌 Project Overview
This project provides a comprehensive business intelligence analysis for **Olist**, the largest department store in Brazilian marketplaces. By processing over 100,000 orders from 2016 to 2018, the analysis uncovers critical insights into customer behavior, logistics efficiency, and revenue trends.

The primary goal is to demonstrate a full data science pipeline: from raw data integration (9 relational tables) to actionable business recommendations.

## 🛠️ Tech Stack
* **Language:** Python 3.12
* **Data Manipulation:** Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn
* **Environment:** Jupyter Notebook / Kaggle

## 🚀 Key Business Insights
* **Revenue Growth:** Identified a significant surge in monthly revenue during late 2017 and early 2018, correlating with platform expansion.
* **Logistics Performance:** Evaluated the delivery lifecycle, identifying key geographical areas with high `freight_ratio` and delivery delays.
* **Payment Preferences:** Analyzed credit card dominance and installments' impact on order value.
* **Customer Satisfaction:** Correlated review scores with delivery time and product categories (e.g., highly-rated niche categories like `cds_dvds_musicals`).

## 📂 Project Structure
* `notebooks/`: Contains the full analytical process with detailed comments.
* `output/`: High-resolution charts and visual summaries of the findings.
* `src/`: Modularized Python scripts for data loading and preprocessing.
